
# 東方旧作風効果音素材

## THPC98-like SoundEffects by PMD

## 概要

東方旧作風効果音素材です。  
PMDを利用して作成しました。  
[ダウンロードはこちら](https://github.com/average34/THPC98SEPMD/releases)

## 使い方

「東方旧作再現効果音wav」フォルダの中に  
効果音のwavが入っています。  
直下のフォルダにはMMLファイル(.mml)とPMDファイル(.M)が入っており、  
それぞれテキストエディタとFMPMDで開くことができます。
